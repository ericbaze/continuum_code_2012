Anaconda Community Edition v.0.8.3

Easy, Scalable Distributed Data Analysis in Python

Anaconda is a distribution that combines the most popular Python packages for data analysis, statistics, and machine learning. It has several tools for a variety of types of cluster computations, including MapReduce batch jobs, interactive parallelism, and MPI. All of the packages in Anaconda are built, tested, and supported by Continuum Analytics. Having a unified runtime for distributed data analysis makes it easier for the broader Python analytics community to share code, examples, and best practices -- without getting tangled in a mess of version numbers and dependencies.

What's Included

The current version of Anaconda is 0.8.3 Public Beta. It is built on Python 2.7.3.

Note: Anaconda 1.0 will feature not only popular, commonly-used open-source packages, but will also include additional libraries from Continuum for fast data access to SQL and NoSQL databases, conventional textfile formats, remote & distributed array processing, and data visualization.

Data Management
  • NumPy - Fast array & tabular data access library
  • Pandas - Statistics-oriented data structures, like R's DataFrame
  • PyTables - Tabular access to HDF5 files
  • h5py - Pythonic, low-level access to HDF5 files
  • PySAM - access to SAM/BAM genome sequence data files
Distributed Computing
  • Disco - An Erlang/Python implementation of MapReduce
  • IPython - interactive parallel Python interpreter kernels
  • MPI4py - Implementation of MPI ptp and collective operations
