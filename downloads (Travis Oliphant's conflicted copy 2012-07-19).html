Installation

  1. After downloading the Anaconda tarball, run the following command from a terminal: 
    sudo tar xpojf anaconda-0.8.0.tar.bz2 -C /opt
  2. Create a Disco user:
    sudo /opt/anaconda/etc/disco/create-disco-user.sh
  3. Add theAanaconda directory to the .bashrc file:
    sudo vim /opt/anaconda/var/disco/.bashrc
    
                                       		 export PATH=/opt/anaconda/bin:$PATH
  4. Change ownership of .bashrc file:
    sudo chown disco:disco  /opt/anaconda/var/disco/.bashrc
  5. Change permissions on Disco account:
    sudo chmod go-w ~disco
  6. Create .ssh directory for Disco keys and set permissions:
    sudo mkdir /opt/anaconda/var/disco/.ssh
    
                                        	sudo chmod go-rwx /opt/anaconda/var/disco/.ssh
    
                                        	sudo chown disco:disco /opt/anaconda/var/disco/.ssh
  7. Start the Disco service
    sudo /opt/anaconda/etc/init.d/disco start

Uninstallation

As all of Anaconda is contained in a single directory, uninstalling Anaconda is simple:

$ rm -rf /opt/anaconda

Reminder: you may need root permissions to execute the above command.
