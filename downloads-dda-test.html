Installation
Anaconda Install Instructions

  1. After downloading the Anaconda tarball, run the following command from a terminal:
    tar xpojf anaconda-0.8.3.tar.bz2 -C/opt
    This will extract all necessary files into the prefix /opt/anaconda.

    You might have to run the above command as root if you do not have permission to write into the /opt directory.

  2. After the extraction is finished, you should add the the anaconda binary directory to your PATH environment variable:
    export PATH=/opt/anaconda/bin:"$PATH"
    You may wish to add the above line to your ~/.bashrc file.

Disco Installation Instructions

  1. After downloading the Anaconda tarball, run the following command from a terminal: 
     tar xpojf anaconda-0.8.3.tar.bz2 -C /opt
  2. Create a Disco user:
     /opt/anaconda/etc/disco/create-disco-user.sh
  3. Add the Anaconda directory to the .bashrc file:
     vim /opt/anaconda/var/disco/.bashrc
    export PATH=/opt/anaconda/bin:$PATH
  4. Change ownership of .bashrc file:
     chown disco:disco /opt/anaconda/var/disco/.bashrc
  5. Change permissions on Disco account:
     chmod go-w ~disco
  6. Create .ssh directory for Disco keys and set permissions:
     mkdir /opt/anaconda/var/disco/.ssh
    
											 chmod go-rwx /opt/anaconda/var/disco/.ssh
    
											 chown disco:disco /opt/anaconda/var/disco/.ssh
  7. Start the Disco service
     /opt/anaconda/etc/init.d/disco start

Uninstalling Anaconda

As all of Anaconda is contained in a single directory, uninstalling Anaconda is simple: 

rm -rf /opt/anaconda
Reminder: you may need root permissions to execute the above command.
