NumbaPro

Array-oriented Python Compiler for NumPy

NumPy is the fundamental array-object of choice for scientists and engineers. It has introduced array-oriented programming techniques to a wide variety of Python programmers. Scientists and engineers are now accustomed to using NumPy's fast and efficient vectorized operations for processing and transforming their data.

NumbaPro allows scientists and engineers to augment NumPy's fast vectorized operations with their own custom Python functions. These functions are compiled (either at run time or build time) from Python expressions to blazing fast native code. Array-oriented programming has always been very fast but sometimes difficult to achieve. With built-in NumPy routines, NumbaPro makes it possible to extend NumPy with compiled Python code.

Features

  • Define custom Python functions which operate on NumPy arrays, and compile them to native code.
  • Computation engine that allows Numba functions to take advantage of multi-core environments.
  • Optimize Numba functions for execution on GPUs.
  • Compile a Python module to a shared library + header file (as if you had programmed in C).

Supported Platforms

  • Linux, Mac OS X, and Windows
  • 32-bit and 64-bit distributions
  • Python 2.7
